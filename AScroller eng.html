class="AScroller" - html element with this class name will be generated by AScroller to scroller with AScrollerBox class name. Arguments will be taken from data-ascroller attribute of source element and width,max-width from style attribue.

    Available arguments of data-ascroller attribute:
gallery - Ascroller size will be match to most big subelement of scroller or size of style attribute. Only one element will be shown in window.
mini-imagesx3 - mini images of AScroller with width in xN elements.
arrows - show arrows of AScroller main window.
condition - show condition of AScroller moving.
full - show full-sized image on click or touch.

Arrows and mini images will not be shown on mobile devices.

Ascroller supports scrolling on PC and mobile devices.


!! Must be supporting display: flex; by browser.



Scroller with following flags: data-ascroller = "gallery mini-imagesx3 arrows condition full", and several subelements




Scroller with following flags: data-ascroller = "gallery condition full", and set width style, and single subelement




Scroller with following flags: data-ascroller = "gallery condition full", and set width style, and several subelements




Scroller with following flags: data-ascroller = "arrows condition full", and several subelements
